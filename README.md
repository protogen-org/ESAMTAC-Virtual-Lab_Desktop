# ESAMTAC Virtual Lab

The ESAMTAC Virtual Lab is the desktop application component of the ESAMTAC online training course. It delivers immersive, scenario-based training exercises that complement the course curriculum.

## Getting Started

To access the full training course and learn more about ESAMTAC, visit:

**[go.esamtac-training.org](https://go.esamtac-training.org)**

## Installation

Download the latest installer from the [Releases](../../releases) page and run `ESAMTAC_Installer.exe`. Follow the on-screen prompts to complete installation.

## Requirements

- Windows 10/11 (64-bit)
- Meta Quest 3 headset (for VR mode)
- Active ESAMTAC course enrollment
